  Key Features                            
                                          
  Intelligent Skill Filtering:            
  - Each summarizer returns has_signal:   
  boolean indicating if actionable data   
  was found                               
  - Skills with no signal are omitted from
   synthesis (prevents "no data" padding) 
  - Only skills that found something      
  meaningful contribute to the report     
                                          
  Conflict Detection:                     
  - Detects contradictions (e.g.,         
  pipeline-waterfall shows zero movement  
  but weekly-recap shows wins)            
  - Orchestrator prompt instructs Claude  
  to use more recent run and note         
  discrepancy                             
                                          
  Structured Output:                      
  - Claude returns JSON with headline,    
  sections, actions,                      
  recommended_next_steps                  
  - Each section has title, content,      
  word_count, source_skills, severity     
  - Actions have urgency levels           
  (today/this_week/this_month) with deal  
  context                                 
                                          
  Word Budget Enforcement:                
  - monday_briefing: 500 words,           
  weekly_business_review: 550, qbr: 600,  
  board_deck: 300                         
  - Tracked in total_word_count field for 
  compliance verification                 
                                          
  Non-Fatal Orchestrator:                 
  - If orchestrator fails, agent falls    
  back to normal synthesis                
  - Ensures delivery continues even if    
  synthesis breaks (logged as warning)
