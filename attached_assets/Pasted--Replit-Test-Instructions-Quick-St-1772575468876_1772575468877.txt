🧪 Replit Test Instructions                
                                             
  Quick Start                                
                                             
  # 1. Set your workspace ID                 
  export                                     
  WORKSPACE_ID="your-workspace-id-here"      
                                             
  # 2. Run the test script                   
  node test-week4-replit.js                  
                                             
  What Gets Tested                           
                                             
  The script automatically tests:            
                                             
  ✅ History List API                        
  - Pagination (limit, offset)               
  - Filtering (skill, status, dates)         
  - Response validation                      
                                             
  ✅ Timeline/Trend Analysis                 
  - 30-day and 7-day trends                  
  - Trend direction                          
  (improving/worsening/stable)               
  - Delta calculations                       
                                             
  ✅ Deal Timeline Tracking                  
  - First flagged date                       
  - Severity changes over time               
  - Recurring issue detection                
                                             
  ✅ Export Functionality                    
  - CSV generation and download              
  - XLSX generation and download             
  - File format validation                   
                                             
  ✅ Error Handling                          
  - Invalid parameters                       
  - Missing data scenarios

  📋 Frontend Testing Checklist              
                                             
  After running the API tests, test the UI:  
                                             
  1. Navigate to History Page                
                                             
  - Click "View History" button in           
  ProactiveBriefing                          
  - Or go to:                                
  http://localhost:5173/investigation/history
                                             
  2. Test Filters                            
                                             
  - Skill dropdown (All Skills, Deal Risk    
  Review, etc.)                              
  - Status dropdown (All Statuses, Completed,
   Failed)                                   
  - Date range pickers (From/To)             
  - Clear Filters button                     
                                             
  3. Timeline Chart                          
                                             
  - Appears when skill is selected           
  - Shows trend indicator (📉 Improving / 📈 
  Worsening / ➡️ Stable)                     
  - 4 lines: At Risk, Critical, Warning,     
  Healthy                                    
  - Hover tooltips work                      
  - Shows run count summary                  
                                             
  4. History Table                           
                                             
  - Column sorting (click headers to sort)   
  - Color-coded badges (severity, status)    
  - CSV/XLSX export buttons                  
  - Row click opens investigation modal      
  - Pagination (Previous/Next buttons)       
                                             
  5. Downloads                               
                                             
  - CSV downloads open in Excel              
  - XLSX has 2 tabs (Summary + Records)      
  - Color-coded cells in XLSX
