skill_runs for chart builder data sourcing

Run these queries against the Neon DB and

Query 1 — What skill runs exist:
SELECT
  skill_id,
  status,
  created_at,
  pg_size_pretty(pg_column_size(result_data))
    as result_data_size,
  pg_size_pretty(pg_column_size(output))
    as output_size
FROM skill_runs
WHERE workspace_id = '4160191d-73bc-414b-97dd-5a1853190378'
  AND status = 'completed'
ORDER BY created_at DESC
LIMIT 10;
